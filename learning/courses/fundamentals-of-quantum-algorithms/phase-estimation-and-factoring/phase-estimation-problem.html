The phase estimation problem

This section of the lesson explains the phase estimation problem. We'll begin with a short discussion of the spectral theorem from linear algebra, and then move on to a statement of the phase estimation problem itself.

Spectral theorem​

The spectral theorem is an important fact from linear algebra that states that matrices of a certain type, called normal matrices, can be expressed in a simple and useful way. We'll only need this theorem for unitary matrices in this lesson, but down the road in this series we'll apply it to Hermitian matrices as well.

Normal matrices​

A square matrix MMM with complex number entries is said to be a normal matrix if it commutes with its conjugate transpose: MM†=M†M.M M^{\dagger} = M^{\dagger} M.MM†=M†M.

Every unitary matrix UUU is normal because

UU†=I=U†U.U U^{\dagger} = \mathbb{I} = U^{\dagger} U.UU†=I=U†U.

Hermitian matrices, which are matrices that equal their own conjugate transpose, are another important class of normal matrices. If HHH is a Hermitian matrix, then

HH†=H2=H†H,H H^{\dagger} = H^2 = H^{\dagger} H,HH†=H2=H†H,

so HHH is normal.

Not every square matrix is normal. For instance, this matrix isn't normal:

(0100)\begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix}(00​10​)

(This is a simple but great example of a matrix that's often very helpful to consider.) It isn't normal because

(0100)(0100)†=(0100)(0010)=(1000)\begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix}^{\dagger} = \begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 0\\ 1 & 0 \end{pmatrix} = \begin{pmatrix} 1 & 0\\ 0 & 0 \end{pmatrix}(00​10​)(00​10​)†=(00​10​)(01​00​)=(10​00​)

while

(0100)†(0100)=(0010)(0100)=(0001).\begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix}^{\dagger} \begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 0\\ 1 & 0 \end{pmatrix} \begin{pmatrix} 0 & 1\\ 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 0\\ 0 & 1 \end{pmatrix}.(00​10​)†(00​10​)=(01​00​)(00​10​)=(00​01​).

Theorem statement​

Now here's a statement of the spectral theorem.

Theorem

Spectral theorem: Let MMM be a normal N×NN\times NN×N complex matrix. There exists an orthonormal basis of NNN-dimensional complex vectors {∣ψ1⟩,…,∣ψN⟩}\bigl\{ \vert\psi_1\rangle,\ldots,\vert\psi_N\rangle \bigr\}{∣ψ1​⟩,…,∣ψN​⟩} along with complex numbers λ1,…,λN\lambda_1,\ldots,\lambda_Nλ1​,…,λN​ such that

M=λ1∣ψ1⟩⟨ψ1∣+⋯+λN∣ψN⟩⟨ψN∣.M = \lambda_1 \vert \psi_1\rangle\langle \psi_1\vert + \cdots + \lambda_N \vert \psi_N\rangle\langle \psi_N\vert.M=λ1​∣ψ1​⟩⟨ψ1​∣+⋯+λN​∣ψN​⟩⟨ψN​∣.

The expression of a matrix in the form

M=∑k=1Nλk∣ψk⟩⟨ψk∣(1)M = \sum_{k = 1}^N \lambda_k \vert \psi_k\rangle\langle \psi_k\vert \tag{1}M=k=1∑N​λk​∣ψk​⟩⟨ψk​∣(1)

is commonly called a spectral decomposition. Notice that if MMM is a normal matrix expressed in the form (1),(1),(1), then the equation

M∣ψj⟩=λj∣ψj⟩M \vert \psi_j \rangle = \lambda_j \vert \psi_j \rangleM∣ψj​⟩=λj​∣ψj​⟩

must be true for every j=1,…,N.j = 1,\ldots,N.j=1,…,N. This is a consequence of the fact that {∣ψ1⟩,…,∣ψN⟩}\bigl\{ \vert\psi_1\rangle,\ldots,\vert\psi_N\rangle \bigr\}{∣ψ1​⟩,…,∣ψN​⟩} is orthonormal:

M∣ψj⟩=(∑k=1Nλk∣ψk⟩⟨ψk∣)∣ψj⟩=∑k=1Nλk∣ψk⟩⟨ψk∣ψj⟩=λj∣ψj⟩M \vert \psi_j \rangle = \left(\sum_{k = 1}^N \lambda_k \vert \psi_k\rangle\langle \psi_k\vert\right)\vert \psi_j\rangle = \sum_{k = 1}^N \lambda_k \vert \psi_k\rangle\langle \psi_k\vert\psi_j \rangle = \lambda_j \vert\psi_j \rangleM∣ψj​⟩=(k=1∑N​λk​∣ψk​⟩⟨ψk​∣)∣ψj​⟩=k=1∑N​λk​∣ψk​⟩⟨ψk​∣ψj​⟩=λj​∣ψj​⟩

That is, each number λj\lambda_jλj​ is an eigenvalue of MMM and ∣ψj⟩\vert\psi_j\rangle∣ψj​⟩ is an eigenvector corresponding to that eigenvalue.

    λ1=1,λ2=1,∣ψ1⟩=∣0⟩,∣ψ2⟩=∣1⟩.\lambda_1 = 1, \hspace{5pt} \lambda_2 = 1, \hspace{5pt} \vert\psi_1\rangle = \vert 0\rangle, \hspace{5pt} \vert\psi_2\rangle = \vert 1\rangle.λ1​=1,λ2​=1,∣ψ1​⟩=∣0⟩,∣ψ2​⟩=∣1⟩.

    I=∣0⟩⟨0∣+∣1⟩⟨1∣.\mathbb{I} = \vert 0\rangle\langle 0\vert + \vert 1\rangle\langle 1\vert.I=∣0⟩⟨0∣+∣1⟩⟨1∣.

As the first example above reveals, there can be some freedom in how eigenvectors are selected. There is, however, no freedom at all in how the eigenvalues are chosen, except for their ordering: the same NNN complex numbers λ1,…,λN,\lambda_1,\ldots,\lambda_N,λ1​,…,λN​, which can include repetitions of the same complex number, will always occur in the equation (1)(1)(1) for a given choice of a matrix M.M.M.

Now let's focus in on unitary matrices. Suppose we have a complex number λ\lambdaλ and a non-zero vector ∣ψ⟩\vert\psi\rangle∣ψ⟩ that satisfy the equation

U∣ψ⟩=λ∣ψ⟩.(2)U\vert\psi\rangle = \lambda\vert\psi\rangle. \tag{2}U∣ψ⟩=λ∣ψ⟩.(2)

That is, λ\lambdaλ is an eigenvalue of UUU and ∣ψ⟩\vert\psi\rangle∣ψ⟩ is an eigenvector corresponding to this eigenvalue.

Unitary matrices preserve Euclidean norm, and so we conclude the following from (2).(2).(2).

∥∣ψ⟩∥=∥U∣ψ⟩∥=∥λ∣ψ⟩∥=∣λ∣∥∣ψ⟩∥\bigl\| \vert\psi\rangle \bigr\| = \bigl\| U \vert\psi\rangle \bigr\| = \bigl\| \lambda \vert\psi\rangle \bigr\| = \vert \lambda \vert \bigl\| \vert\psi\rangle \bigr\|​∣ψ⟩​=​U∣ψ⟩​=​λ∣ψ⟩​=∣λ∣​∣ψ⟩​

The condition that ∣ψ⟩\vert\psi\rangle∣ψ⟩ is non-zero implies that ∥∣ψ⟩∥≠0,\bigl\| \vert\psi\rangle \bigr\|\neq 0,​∣ψ⟩​=0, so we can cancel it from both sides to obtain

∣λ∣=1.\vert \lambda \vert = 1.∣λ∣=1.

This reveals that eigenvalues of unitary matrices must always have absolute value equal to one, so they lie on the unit circle.

T={α∈C:∣α∣=1}\mathbb{T} = \{\alpha\in\mathbb{C} : \vert\alpha\vert = 1\}T={α∈C:∣α∣=1}

(The symbol T\mathbb{T}T is a common name for the complex unit circle. The name S1S^1S1 is also common.)

Phase estimation problem statement​

In the phase estimation problem, we're given a quantum state ∣ψ⟩\vert \psi\rangle∣ψ⟩ of nnn qubits, along with a unitary quantum circuit that acts on nnn qubits. We're promised that ∣ψ⟩\vert \psi\rangle∣ψ⟩ is an eigenvector of the unitary matrix UUU that describes the action of the circuit, and our goal is to compute or approximate the eigenvalue λ\lambdaλ to which ∣ψ⟩\vert \psi\rangle∣ψ⟩ corresponds. More precisely, because λ\lambdaλ lies on the complex unit circle, we can write

λ=e2πiθ\lambda = e^{2\pi i \theta}λ=e2πiθ

for a unique real number θ\thetaθ satisfying 0≤θ<1.0\leq\theta<1.0≤θ<1. The goal of the problem is to compute or approximate this real number θ.\theta.θ.

Phase estimation problem

Input: A unitary quantum circuit for an nnn-qubit operation UUU along with an nnn-qubit quantum state ∣ψ⟩\vert\psi\rangle∣ψ⟩
Promise: ∣ψ⟩\vert\psi\rangle∣ψ⟩ is an eigenvector of UUU
Output: an approximation to the number θ∈[0,1)\theta\in[0,1)θ∈[0,1) satisfying U∣ψ⟩=e2πiθ∣ψ⟩U\vert\psi\rangle = e^{2\pi i \theta}\vert\psi\rangleU∣ψ⟩=e2πiθ∣ψ⟩

Here are a few remarks about this problem statement:

  1. The phase estimation problem is different from other problems we've seen so far in the course in that the input includes a quantum state. Typically we focus on problems having classical inputs and outputs, but nothing prevents us from considering quantum state inputs like this. In terms of its practical relevance, the phase estimation problem is typically encountered as a subproblem inside of a larger computation, like we'll see in the context of integer factorization later in the lesson.

  2. The statement of the phase estimation problem above isn't specific about what constitutes an approximation of θ,\theta,θ, but we can formulate more precise problem statements depending on our needs and interests. In the context of integer factorization, we'll demand a very precise approximation to θ,\theta,θ, but in other cases we might be satisfied with a very rough approximation. We'll discuss shortly how the precision we require affects the computational cost of a solution.

  3. Notice that as we go from θ=0\theta = 0θ=0 toward θ=1\theta = 1θ=1 in the phase estimation problem, we're going all the way around the unit circle, starting from e2πi⋅0=1e^{2\pi i \cdot 0} = 1e2πi⋅0=1 and moving counter-clockwise toward e2πi⋅1=1.e^{2\pi i \cdot 1} = 1.e2πi⋅1=1. That is, when we reach θ=1\theta = 1θ=1 we're back where we started at θ=0.\theta = 0.θ=0. So, as we consider the accuracy of approximations, choices of θ\thetaθ near 111 should be considered as being near 0.0.0. For example, an approximation θ=0.999\theta = 0.999θ=0.999 should be considered as being within 1/10001/10001/1000 of θ=0.\theta = 0.θ=0.
