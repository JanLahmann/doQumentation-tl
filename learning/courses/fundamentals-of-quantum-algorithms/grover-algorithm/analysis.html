Now we'll analyze Grover's algorithm to understand how it works. We'll start with what could be described as a symbolic analysis, where we calculate how the Grover operation GGG acts on certain states, and then we'll tie this symbolic analysis to a geometric picture that's helpful for visualizing how the algorithm works.

Solutions and non-solutions​

Let's start by defining two sets of strings.

A0={x∈Σn:f(x)=0}A1={x∈Σn:f(x)=1}\begin{aligned} A_0 &= \bigl\{ x\in\Sigma^n : f(x) = 0\bigr\} \\ A_1 &= \bigl\{ x\in\Sigma^n : f(x) = 1\bigr\} \end{aligned}A0​A1​​={x∈Σn:f(x)=0}={x∈Σn:f(x)=1}​

The set A1A_1A1​ contains all of the solutions to our search problem while A0A_0A0​ contains the strings that aren't solutions (which we can refer to as non-solutions when it's convenient). These two sets satisfy A0∩A1=∅A_0 \cap A_1 = \varnothingA0​∩A1​=∅ and A0∪A1=Σn,A_0 \cup A_1 = \Sigma^n,A0​∪A1​=Σn, which is to say that this is a bipartition of Σn.\Sigma^n.Σn.

Next we'll define two unit vectors representing uniform superpositions over the sets of solutions and non-solutions.

∣A0⟩=1∣A0∣∑x∈A0∣x⟩∣A1⟩=1∣A1∣∑x∈A1∣x⟩\begin{aligned} \vert A_0\rangle &= \frac{1}{\sqrt{\vert A_0\vert}} \sum_{x\in A_0} \vert x\rangle \\ \vert A_1\rangle &= \frac{1}{\sqrt{\vert A_1\vert}} \sum_{x\in A_1} \vert x\rangle \end{aligned}∣A0​⟩∣A1​⟩​=∣A0​∣​1​x∈A0​∑​∣x⟩=∣A1​∣​1​x∈A1​∑​∣x⟩​

Formally speaking, each of these vectors is only defined when its corresponding set is nonempty, but hereafter we're going to focus on the case that neither A0A_0A0​ nor A1A_1A1​ is empty. The cases that A0=∅A_0 = \varnothingA0​=∅ and A1=∅A_1 = \varnothingA1​=∅ are easily handled separately, and we'll do that later.

As an aside, the notation being used here is common: any time we have a finite and nonempty set S,S,S, we can write ∣S⟩\vert S\rangle∣S⟩ to denote the quantum state vector that's uniform over the elements of S.S.S.

Let's also define ∣u⟩\vert u \rangle∣u⟩ to be a uniform quantum state over all nnn-bit strings:

∣u⟩=1N∑x∈Σn∣x⟩.\vert u\rangle = \frac{1}{\sqrt{N}} \sum_{x\in\Sigma^n} \vert x\rangle.∣u⟩=N​1​x∈Σn∑​∣x⟩.

∣u⟩=∣A0∣N∣A0⟩+∣A1∣N∣A1⟩.\vert u\rangle = \sqrt{\frac{\vert A_0 \vert}{N}} \vert A_0\rangle + \sqrt{\frac{\vert A_1 \vert}{N}} \vert A_1\rangle.∣u⟩=N∣A0​∣​​∣A0​⟩+N∣A1​∣​​∣A1​⟩.

We also have that ∣u⟩=H⊗n∣0n⟩,\vert u\rangle = H^{\otimes n} \vert 0^n \rangle,∣u⟩=H⊗n∣0n⟩, so ∣u⟩\vert u\rangle∣u⟩ represents the state of the register Q\mathsf{Q}Q after the initialization in step 1 of Grover's algorithm.

This implies that just before the iterations of GGG happen in step 2, the state of Q\mathsf{Q}Q is contained in the two-dimensional vector space spanned by ∣A0⟩\vert A_0\rangle∣A0​⟩ and ∣A1⟩,\vert A_1\rangle,∣A1​⟩, and moreover the coefficients of these vectors are real numbers. As we will see, the state of Q\mathsf{Q}Q will always have these properties — meaning that the state is a real linear combination of ∣A0⟩\vert A_0\rangle∣A0​⟩ and ∣A1⟩\vert A_1\rangle∣A1​⟩ — after any number of iterations of the operation GGG in step 2.

An observation about the Grover operation​

We'll now turn our attention to the Grover operation

Notice that

(−1)g(x)=(−1)1⊕f(x)=−(−1)f(x)(-1)^{g(x)} = (-1)^{1 \oplus f(x)} = - (-1)^{f(x)}(−1)g(x)=(−1)1⊕f(x)=−(−1)f(x)

for every string x∈Σn,x\in\Sigma^n,x∈Σn, and therefore

Zg=−Zf.Z_g = - Z_f.Zg​=−Zf​.

This means that if we were to substitute the function fff with the function g,g,g, Grover's algorithm wouldn't function any differently — because the states we obtain from the algorithm in the two cases are necessarily equivalent up to a global phase.

This isn't a problem! Intuitively speaking, the algorithm doesn't care which strings are solutions and which are non-solutions — it only needs to be able to distinguish solutions and non-solutions to operate correctly.

Geometric picture​

Now let's connect the analysis we just went through to a geometric picture. The idea is that the operation GGG is the product of two reflections, ZfZ_fZf​ and H⊗nZORH⊗n.H^{\otimes n} Z_{\mathrm{OR}} H^{\otimes n}.H⊗nZOR​H⊗n. And the net effect of performing two reflections is to perform a rotation.

Let's start with Zf.Z_f.Zf​. As we already observed previously, we have

Zf∣A0⟩=∣A0⟩Zf∣A1⟩=−∣A1⟩.\begin{aligned} Z_f \vert A_0\rangle & = \vert A_0\rangle \\[1mm] Z_f \vert A_1\rangle & = -\vert A_1\rangle. \end{aligned}Zf​∣A0​⟩Zf​∣A1​⟩​=∣A0​⟩=−∣A1​⟩.​

Within the two-dimensional vector space spanned by ∣A0⟩\vert A_0\rangle∣A0​⟩ and ∣A1⟩,\vert A_1\rangle,∣A1​⟩, this is a reflection about the line parallel to ∣A0⟩,\vert A_0\rangle,∣A0​⟩, which we'll call L1.L_1.L1​. Here's a figure illustrating the action of this reflection on a hypothetical unit vector ∣ψ⟩,\vert\psi\rangle,∣ψ⟩, which we're assuming is a real linear combination of ∣A0⟩\vert A_0\rangle∣A0​⟩ and ∣A1⟩.\vert A_1\rangle.∣A1​⟩.

Second we have the operation H⊗nZORH⊗n,H^{\otimes n} Z_{\mathrm{OR}} H^{\otimes n},H⊗nZOR​H⊗n, which we've already seen can be written as
