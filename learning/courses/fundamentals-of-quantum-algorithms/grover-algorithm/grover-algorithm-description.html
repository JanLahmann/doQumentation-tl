Description of Grover's algorithm

In this section, we'll describe Grover's algorithm. We'll begin by discussing phase query gates and how to build them, followed by the description of Grover's algorithm itself. Finally, we'll briefly discuss how this algorithm is naturally applied to searching.

Phase query gates​

Grover's algorithm makes use of operations known as phase query gates. In contrast to an ordinary query gate Uf,U_f,Uf​, defined for a given function fff in the usual way described previously, a phase query gate for the function fff is defined as

This implementation makes use of the phase kickback phenomenon, and requires that one workspace qubit, initialized to a ∣−⟩\vert -\rangle∣−⟩ state, is made available. This qubit remains in the ∣−⟩\vert - \rangle∣−⟩ state after the implementation has completed, and can be reused (to implement subsequent ZfZ_fZf​ gates, for instance) or simply discarded.

In addition to the operation Zf,Z_f,Zf​, we will also make use of a phase query gate for the nnn-bit OR function, which is defined as follows for each string x∈Σn.x\in\Sigma^n.x∈Σn.

Description of the algorithm​

Now that we have the two operations ZfZ_fZf​ and ZOR,Z_{\mathrm{OR}},ZOR​, we can describe Grover's algorithm.

The algorithm refers to a number t,t,t, which is the number of iterations it performs (and therefore the number of queries to the function fff it requires). This number ttt isn't specified by Grover's algorithm as we're describing it, and we'll discuss later in the lesson how it can be chosen.

Grover's algorithm
  1. Initialize an nnn qubit register Q\mathsf{Q}Q to the all-zero state ∣0n⟩\vert 0^n \rangle∣0n⟩ and then apply a Hadamard operation to each qubit of Q.\mathsf{Q}.Q.

  3. Measure the qubits of Q\mathsf{Q}Q with respect to standard basis measurements and output the resulting string.

Application to search​

Grover's algorithm can be applied to the Search problem as follows:

  • Choose the number ttt in step 2. (This is discussed later in the lesson.)
  • Run Grover's algorithm on the function f,f,f, using whatever choice we made for t,t,t, to obtain a string x∈Σn.x\in\Sigma^n.x∈Σn.
  • Query the function fff on the string xxx to see if it's a valid solution:
    • If f(x)=1,f(x) = 1,f(x)=1, then we have found a solution, so we can stop and output x.x.x.
    • Otherwise, if f(x)=0,f(x) = 0,f(x)=0, then we can either run the procedure again, possibly with a different choice for t,t,t, or we can decide to give up and output "no solution."

Once we've analyzed how Grover's algorithm works, we'll see that by taking t=O(N),t = O(\sqrt{N}),t=O(N​), we obtain a solution to our search problem (if one exists) with high probability.
