Equivalence of the representations

Hindi pa naisalin

Ang pahinang ito ay hindi pa naisalin. Nakikita mo ang orihinal na bersyon sa Ingles.

We've now discussed three different ways to represent channels in mathematical terms, namely Stinespring representations, Kraus representations, and Choi representations. We also have the definition of a channel, which states that a channel is a linear mapping that always transforms density matrices into density matrices, even when the channel is applied to just part of a compound system. The remainder of the lesson is devoted to a mathematical proof that the three representations are equivalent and precisely capture the definition.

Overview of the proof​

Our goal is to establish the equivalence of a collection of four statements, and we'll begin by writing them down precisely. All four statements follow the same conventions that have been used throughout the lesson, namely that Φ\PhiΦ is a linear mapping from square matrices to square matrices, the rows and columns of the input matrices have been placed in correspondence with the classical states of a system X\mathsf{X}X (the input system), and the rows and columns of the output matrices have been placed in correspondence with the classical states of a system Y\mathsf{Y}Y (the output system).

  1. Φ\PhiΦ is a channel from X\mathsf{X}X to Y.\mathsf{Y}.Y. That is, Φ\PhiΦ always transforms density matrices to density matrices, even when it acts on one part of a larger compound system.

  2. The Choi matrix J(Φ)J(\Phi)J(Φ) is positive semidefinite and satisfies the condition Tr⁡Y(J(Φ))=IX.\operatorname{Tr}_{\mathsf{Y}}(J(\Phi)) = \mathbb{I}_{\mathsf{X}}.TrY​(J(Φ))=IX​.

  3. There is a Kraus representation for Φ.\Phi.Φ. That is, there exist matrices A0,…,AN−1A_0,\ldots,A_{N-1}A0​,…,AN−1​ for which the equation Φ(ρ)=∑k=0N−1AkρAk†\Phi(\rho) = \sum_{k = 0}^{N-1} A_k \rho A_k^{\dagger}Φ(ρ)=∑k=0N−1​Ak​ρAk†​ is true for every input ρ,\rho,ρ, and that satisfy the condition ∑k=0N−1Ak†Ak=IX.\sum_{k = 0}^{N-1} A_k^{\dagger} A_k = \mathbb{I}_{\mathsf{X}}.∑k=0N−1​Ak†​Ak​=IX​.

  4. There is a Stinespring representation for Φ.\Phi.Φ. That is, there exist systems W\mathsf{W}W and G\mathsf{G}G for which the pairs (W,X)(\mathsf{W},\mathsf{X})(W,X) and (G,Y)(\mathsf{G},\mathsf{Y})(G,Y) have the same number of classical states, along with a unitary matrix UUU representing a unitary operation from (W,X)(\mathsf{W},\mathsf{X})(W,X) to (G,Y),(\mathsf{G},\mathsf{Y}),(G,Y), such that Φ(ρ)=Tr⁡G(U(∣0⟩⟨0∣⊗ρ)U†).\Phi(\rho) = \operatorname{Tr}_{\mathsf{G}}\bigl( U (\vert 0\rangle\langle 0 \vert \otimes \rho) U^{\dagger} \bigr).Φ(ρ)=TrG​(U(∣0⟩⟨0∣⊗ρ)U†).

The way the proof works is that a cycle of implications is proved: the first statement in our list implies the second, the second implies the third, the third implies the fourth, and the fourth statement implies the first. This establishes that all four statements are equivalent — which is to say that they're either all true or all false for a given choice of Φ\PhiΦ — because the implications can be followed transitively from any one statement to any other.

This is a common strategy when proving that a collection of statements are equivalent, and a useful trick to use in such a context is to set up the implications in a way that makes them as easy to prove as possible. That is the case here — and in fact we've already encountered two of the four implications.

Choi to Kraus representations​

Stinespring representations back to the definition​

The final implication is 4 ⇒\Rightarrow⇒ 1. That is, we assume that we have a unitary operation transforming a pair of systems (W,X)(\mathsf{W},\mathsf{X})(W,X) into a pair (G,Y),(\mathsf{G},\mathsf{Y}),(G,Y), and our goal is to conclude that the mapping

Φ(ρ)=Tr⁡G(U(∣0⟩⟨0∣W⊗ρ)U†)\Phi(\rho) = \operatorname{Tr}_{\mathsf{G}} \bigl( U (\vert 0\rangle \langle 0 \vert_{\mathsf{W}} \otimes \rho)U^{\dagger}\bigr)Φ(ρ)=TrG​(U(∣0⟩⟨0∣W​⊗ρ)U†)

is a valid channel. From its form, it is evident that Φ\PhiΦ is linear, and it remains to verify that it always transforms density matrices into density matrices. This is pretty straightforward and we've already discussed the key points.

In particular, if we start with a density matrix σ\sigmaσ of a compound system (Z,X),(\mathsf{Z},\mathsf{X}),(Z,X), and then add on an additional workspace system W,\mathsf{W},W, we will certainly be left with a density matrix. If we reorder the systems (W,Z,X)(\mathsf{W},\mathsf{Z},\mathsf{X})(W,Z,X) for convenience, we can write this state as

∣0⟩⟨0∣W⊗σ.\vert 0\rangle\langle 0\vert_{\mathsf{W}} \otimes \sigma.∣0⟩⟨0∣W​⊗σ.

We then apply the unitary operation U,U,U, and as we already discussed this is a valid channel, and hence maps density matrices to density matrices. Finally, the partial trace of a density matrix is another density matrix.

Another way to say this is to observe first that each of these things is a valid channel:

  1. Introducing an initialized workspace system.
  2. Performing a unitary operation.
  3. Tracing out a system.

And finally, any composition of channels is another channel — which is immediate from the definition, but is also a fact worth observing in its own right.

This completes the proof of the final implication, and therefore we've established the equivalence of the four statements listed at the start of the section.
